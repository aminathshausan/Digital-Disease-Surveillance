# Digital-Disease-Surveillance

This repository makes available the source code for the manuscript: 
"Enhancing influenza surveillance in Australia using Google search data combined with machine learning".
The computation concerns model fits and forecast validations for 4 types of machine learning models: 
elastic net, support vector, random forest, and feedforward neural network regressions. 
All codes are implemented using the Python software (>=3.14.12).

# Directory structure
```
├── project
│   ├── data
│   ├── src

- The `data` folder contains all processed data required to run the codes in the `src` folder. 
- The `src` folder contains all source codes required to produce the results and images in the manuscript.
